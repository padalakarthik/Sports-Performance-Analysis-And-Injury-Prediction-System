Help & Documentation

Step-by-Step Guide

  1. Log In: Use your username and password to access the dashboard. Sign up if you don't have an account.
  2. Search for a Player: Enter a player's name (e.g., "L.Messi") in the search bar to view their data, injury predictions, and recommendations.
  3. Update Data (Coaches/Admins): Input real-time training load and heart rate to get updated predictions and recommendations.
  4. Review Recommendations: Check the yellow "Real-Time Recommendations" section for actionable advice based on player data.
  5. Analyze Graphs: View visualizations like "Training Load vs Heart Rate" and "Historical Performance Trends" to understand trends.
  6. Manage Settings (Coaches/Admins): Go to the Settings page to add/delete users or customize graph visibility.

Roles and Permissions

- Users: View player data and recommendations.
- Coaches: Update player data and access settings.
- Admins: Full access, including user management.
